# Skill: Social Media Content

## When to use
User asks for social media posts, captions, content calendar, or platform-specific content.

## Platform Specs

### Instagram
- Caption: 125-150 words for engagement, up to 2200 chars allowed
- Hashtags: 5-10 relevant ones, mix of broad + niche
- Image: 1080x1080 (feed), 1080x1920 (story/reel)
- Hook in first line (shows before "...more")
- Line breaks for readability

### Facebook
- Post: 40-80 words get highest engagement
- Link posts: short teaser + link
- Image: 1200x630 (shared), 1080x1080 (feed)
- Questions and polls drive comments

### TikTok
- Caption: under 150 chars (shows on screen)
- Hook in first 3 seconds of video concept
- Trending sounds/formats when relevant
- Hashtags: 3-5 max

### LinkedIn
- Post: 150-300 words, professional but personal
- First line is the hook (shows before "...see more")
- Use line breaks and emojis sparingly
- Stories > stats (but stats support stories)

### X/Twitter
- 280 chars max, aim for 100-200 for engagement
- Thread format for longer content (hook tweet → details)
- No hashtag spam — 1-2 max

## Content Pillars
For any client, rotate between:
1. **Educational** (40%) — teach something useful
2. **Behind the scenes** (20%) — humanize the brand
3. **Social proof** (20%) — testimonials, results, wins
4. **Promotional** (15%) — offers, CTAs, events
5. **Entertainment** (5%) — memes, trends, fun

## Rules
- Write in the client's voice, not generic marketing speak
- Every post needs a hook in the first line
- Include a CTA on promotional posts
- For calendars: plan a full week (7 posts minimum)
- Generate image prompts for each post using image_generate

## Output
Save as artifact. For calendars, use a markdown table format.
