# Skill: Blog Post Writing

## When to use
User asks you to write a blog post, article, or long-form content.

## Structure
1. **Hook** (1-2 sentences) — grab attention with a bold statement, question, or story
2. **Context** (1 paragraph) — why this matters to the reader
3. **Body** (3-5 sections with H2 headers) — the meat, each section 2-3 paragraphs
4. **Call to Action** (1 paragraph) — what the reader should do next

## Rules
- Write in the client's brand voice (check their profile/memory for tone)
- Use short paragraphs (2-3 sentences max)
- Include at least one story or example per section
- Write at an 8th grade reading level — clear, not dumbed down
- NO corporate jargon: "leverage", "synergy", "paradigm" are banned
- Start with the most interesting point, not background
- Every H2 should make the reader want to read that section
- End paragraphs on strong words, not weak qualifiers
- Target 800-1200 words for standard posts, 1500-2000 for pillar content

## SEO Basics
- Include the target keyword in: title, first paragraph, one H2, last paragraph
- Use related keywords naturally throughout
- Write a meta description (155 chars) at the end
- Suggest 3-5 internal link opportunities if you know the client's other content

## Output
Always save as an artifact using create_artifact with a descriptive filename like "blog-youth-empowerment-summer-camp.md"
