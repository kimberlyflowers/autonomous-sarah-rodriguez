# Skill: New Client Onboarding

## When to use
When setting up a new client/business in BLOOM, or when the user mentions onboarding, setup, or getting started with a new account.

## Onboarding Checklist

### Phase 1: Discovery (Day 1)
- [ ] Business name, industry, website URL
- [ ] Primary contact name, email, phone
- [ ] What services/products do they offer?
- [ ] Who is their target audience?
- [ ] What are their top 3 business goals?
- [ ] Current marketing: what are they doing now?
- [ ] Pain points: what's not working?
- [ ] Brand voice: formal, casual, bold, friendly?
- [ ] Brand colors and logo (ask them to share)

### Phase 2: GHL Setup (Day 1-2)
- [ ] Create GHL sub-account
- [ ] Import existing contacts (CSV if they have one)
- [ ] Set up pipeline stages
- [ ] Connect their domain for funnels/websites
- [ ] Set up email sending domain (SPF/DKIM)
- [ ] Create welcome automation workflow
- [ ] Set up appointment calendar

### Phase 3: Content Foundation (Day 2-5)
- [ ] Write welcome email sequence (3 emails)
- [ ] Create 2 social media templates
- [ ] Write first blog post
- [ ] Design email header/banner with brand colors
- [ ] Set up social media posting schedule

### Phase 4: Launch (Day 5-7)
- [ ] Test all automations
- [ ] Send test email/SMS
- [ ] Review pipeline with client
- [ ] Schedule first week of content
- [ ] Set up daily lead check task

## Questions to Ask
1. "What's the #1 thing you want Sarah to handle for you daily?"
2. "How do you currently follow up with new leads?"
3. "What's your biggest marketing frustration right now?"
4. "Who's your ideal customer? Describe them in detail."
5. "What would make you say 'this AI employee was worth every penny'?"

## Output
Create a client profile document saved as an artifact.
Log all onboarding actions to the activity log.
