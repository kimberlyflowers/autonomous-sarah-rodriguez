# Image Generation — Professional Visual Content

You have access to Gemini 2.5 Flash Image (Nano Banana) for generating images. This skill teaches you how to write prompts that produce **professional-quality results** instead of generic AI output.

## Core Principle: Specificity Over Generality

**Bad prompt:** "A coffee shop"
**Good prompt:** "A cozy neighborhood coffee shop interior, morning sunlight streaming through large windows, warm wood furniture, exposed brick walls, professional architectural photography, 35mm lens, natural lighting, shot on Canon 5D"

The difference: **photographic style, lighting, composition, mood, technical details**.

## Image Types & Prompting Formulas

### 1. FLYERS & POSTERS (Event Marketing, Promotions)

**Formula:** [Event Type] + [Visual Style] + [Layout] + [Typography] + [Color Mood]

**Prompt Template:**
```
A [bold/minimalist/vintage] [event type] flyer design. 
[Primary visual element - photo/illustration/pattern].
[Typography style and hierarchy].
[Color palette and mood].
Professional graphic design, high contrast, print-ready, 300 DPI quality.
Aspect ratio: vertical poster format.
```

**Example - Yoga Workshop Flyer:**
```
A serene, minimalist yoga workshop flyer design. 
Centered composition featuring a silhouette of a woman in tree pose against a soft gradient sunrise (coral to lavender). 
Large, elegant serif title "MORNING FLOW" at top, clean sans-serif body text below.
Muted pastel color palette: blush pink, sage green, cream.
Professional graphic design, balanced negative space, Instagram-worthy aesthetic, print-ready quality.
```

**Example - Music Festival Poster:**
```
A bold, energetic music festival poster design.
Vibrant abstract wave patterns in neon colors (electric blue, hot pink, lime green) flowing across the composition.
Thick, geometric sans-serif title dominating the top third, artist lineup in condensed font below.
Dark background with glowing accents, modern club aesthetic.
Professional graphic design, high-impact visuals, suitable for both print and digital.
```

## Advanced Techniques

### Aspect Ratios for Different Uses
- **Flyers/Posters:** Vertical (portrait) - 2:3 or 3:4
- **Social Media:** Square 1:1 (Instagram), vertical 4:5 (Stories), horizontal 16:9 (YouTube)
- **Website Heroes:** Wide 16:9 or ultra-wide 21:9
- **Product Photos:** Square 1:1 or 4:5
- **Infographics:** Vertical 2:3 or 3:4 (easy scrolling)

### Style References That Work Well
- **Photography Styles:** "shot on Canon 5D", "Hasselblad medium format", "35mm film", "professional DSLR"
- **Lighting:** "golden hour", "studio softbox", "dramatic rim lighting", "natural window light", "moody low-key"
- **Composition:** "rule of thirds", "centered symmetry", "diagonal leading lines", "shallow depth of field", "negative space"
- **Mood Words:** serene, energetic, luxurious, minimal, bold, warm, cool, intimate, dramatic, playful

### What NOT to Do
- ❌ Vague prompts: "make it look good" 
- ❌ Generic requests: "a nice background"
- ❌ Missing technical details: forgetting to specify aspect ratio, lighting, style
- ❌ Overcomplicating: trying to fit too many elements in one image

## Workflow

1. **Understand the use case:** Flyer? Social post? Website hero? Product shot?
2. **Choose the right formula** from above
3. **Write a detailed prompt** with: subject, style, lighting, mood, technical specs
4. **Specify aspect ratio** based on where it will be used
5. **Call image_generate** with your prompt and size
6. **Embed the returned image URL** in the deliverable (HTML, doc, etc.)
