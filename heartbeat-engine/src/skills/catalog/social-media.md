# Social Media Content

## Platform-Specific Guidelines

### Instagram
- Caption: Hook in first line (before "...more"), 2200 char max
- Hashtags: 5-15 relevant, mix of volume (500K+, 50K+, 10K+)
- Format: Short paragraphs with line breaks, emojis as bullets
- CTA: "Save this for later" / "Share with someone who needs this"
- Best times: Tue-Fri, 10am-2pm

### TikTok
- Hook in first 3 seconds (text overlay or spoken)
- Caption: Short, punchy, include 3-5 hashtags
- Trending sounds > original audio for reach
- Ideal length: 30-60 seconds for retention
- Post consistently: 1-3x daily

### Facebook
- Longer form OK, storytelling works
- Questions drive comments
- Native video > linked video
- Groups > Pages for engagement
- Best times: Wed-Fri, 1pm-4pm

### LinkedIn
- Professional but human
- First line is everything (hook before fold)
- Use line breaks for readability
- Personal stories + business lessons perform best
- Hashtags: 3-5 max, professional
- Best times: Tue-Thu, 8am-10am

## Content Pillars (adapt per client)
1. Educational (teach something)
2. Inspirational (motivate/uplift)
3. Behind-the-scenes (humanize)
4. Social proof (testimonials, results)
5. Promotional (offers, CTAs — max 20% of posts)

## Output Format
Always deliver as create_artifact with:
- Platform-specific version of each post
- Suggested posting time
- Hashtag set
- Image/video description for generation
- CTA suggestion
