# Blog & Long-Form Content

## Structure
Every blog post follows this skeleton:
1. Hook (first 2 sentences grab attention — stat, question, or bold claim)
2. Context (why this matters to the reader)
3. Main content (H2 sections, each a standalone value unit)
4. Takeaway / CTA (what to do next)

## SEO Essentials
- Title: Include primary keyword, under 60 characters
- Meta description: 150-160 chars, includes keyword + CTA
- H2s every 200-300 words
- Internal links to other client content
- External links to authoritative sources
- Alt text on all images
- Target 1,200-2,000 words for ranking

## Voice & Tone
- Write at 8th grade reading level
- Short paragraphs (2-3 sentences max)
- Use "you" more than "we"
- Active voice always
- No jargon unless the audience expects it
- Match the client's brand voice from their existing content

## Content Types
- **How-to guides**: Step-by-step, numbered, screenshot-ready
- **Listicles**: Odd numbers perform better (7, 11, 15)
- **Thought leadership**: Opinion + data + story
- **Case studies**: Problem → Solution → Results with numbers
- **FAQ posts**: Answer real questions from the audience

## Output Format
Always deliver as create_artifact with:
- Title + meta description
- Full blog post in Markdown
- 3 social media captions (LinkedIn, Facebook, Instagram)
- Suggested featured image description for image generation
